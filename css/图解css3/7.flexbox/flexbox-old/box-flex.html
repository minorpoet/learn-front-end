box-flex: number 灵活控制伸缩项目在伸缩容器中的显示空间；
1. 根据在伸缩项目上设置的box-flex数值总和，将容器内的额外空间分成n等分，然后伸缩项目扩展相应份
2. 当容器宽度小于伸缩项目的宽度之和时，则反向将溢出的空间分成n等分，然后伸缩项目收缩相应份

1. 扩展box-flex:1; box-flex:1;

我是一个绿色的伸缩项目。

我是一个橙色的伸缩项目，我是用来测试box-flex的使用的。

2. 扩展box-flex:1;

我是一个绿色的伸缩项目。

我是一个橙色的伸缩项目，我是用来测试box-flex的使用的。

3. 扩展 box-flex:1; box-flex:2;

我是一个绿色的伸缩项目。

我是一个橙色的伸缩项目，我是用来测试box-flex的使用的。

4. 收缩 box-flex:1; box-flex:1;

我是一个绿色的伸缩项目。

我是一个橙色的伸缩项目，我是用来测试box-flex的使用的。
